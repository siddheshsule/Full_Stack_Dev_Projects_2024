CSS Cheatsheet


Property Description Example
Color & Background
color Sets the color of text color: #333;
background-color Sets the background color of an element background-color: #f0f0f0;
background-image Sets a background image for an element background-image: url('image.jpg');
background-repeat Sets if/how a background image will be repeated background-repeat: no-repeat;
background-size Specifies the size of the background image background-size: cover;
opacity Sets the opacity/transparency of an element opacity: 0.5;
Text & Font
font-family Specifies the font of text font-family: Arial, sans-serif;
font-size Sets the size of the font font-size: 16px;
font-weight Sets the thickness of the text font-weight: bold;
text-align Aligns text horizontally text-align: center;
text-decoration Adds decoration to text (e.g., underline) text-decoration: underline;
line-height Sets the height of lines of text line-height: 1.5;
letter-spacing Adjusts the spacing between characters letter-spacing: 2px;
text-transform Controls the capitalization of text text-transform: uppercase;
white-space Controls how whitespace is handled white-space: nowrap;
Box Model
margin Sets the space outside the element margin: 20px;
padding Sets the space inside the element, between the content and border padding: 10px;
border Sets the border around an element border: 1px solid #ccc;
border-radius Rounds the corners of an element border-radius: 5px;
width Sets the width of an element width: 100px;
height Sets the height of an element height: 200px;
box-shadow Applies shadow to an element box-shadow: 2px 2px 5px rgba(0,0,0,0.3);
display Specifies the display behavior of an element display: inline-block;
overflow Controls what happens to content that overflows an element's box overflow: hidden;
Positioning
position Specifies the positioning method of an element position: absolute;
top, right, bottom, left Offsets the element from its nearest positioned ancestor top: 10px; left: 20px;
z-index Specifies the stack order of an element z-index: 1000;
float Floats an element to the left or right float: right;
clear Specifies which sides of an element should not be next to floating elements clear: both;
Flexbox
display: flex Enables flexbox layout display: flex;
flex-direction Specifies the direction of the flex items flex-direction: row;
justify-content Aligns items horizontally in the flex container justify-content: space-between;
align-items Aligns items vertically in the flex container align-items: center;
flex-wrap Controls whether flex items are forced into a single line or can wrap onto multiple lines flex-wrap: wrap;
flex Specifies how a flex item will grow or shrink flex: 1;
Grid
display: grid Enables grid layout display: grid;
grid-template-columns Defines the number and sizes of columns in the grid grid-template-columns: 1fr 2fr;
grid-template-rows Defines the number and sizes of rows in the grid grid-template-rows: 100px auto;
grid-gap Sets the gap between grid items grid-gap: 20px;
grid-area Assigns a grid item to a specific area grid-area: header;
justify-items Aligns grid items along the inline (row) axis justify-items: start;
align-items Aligns grid items along the block (column) axis align-items: stretch;
Transitions & Animations
transition Specifies the transition between states of an element transition: all 0.3s ease;
animation Applies an animation to an element animation: slide 2s infinite;
@keyframes Defines the keyframes for an animation @keyframes slide { from {left: 0;} to {left: 100%;}}
